# Shopping Cart System

## Description
A simple Shopping Cart System developed using Core Java. This console application allows users to:
- View products
- Add products to the cart
- View the shopping cart
- Checkout

## Features
- View Products
- Add Product to Cart
- View Cart
- Checkout
- Exit

## Technologies Used
- Java
- VS Code
